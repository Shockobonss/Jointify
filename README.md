# Jointify
HI!!
This is my first Blender addon called Jointify. It is useful to clean up visual bone clutter that we get with skeletons from Unreal Engine
My goal was to get as close as possible to the Unreal Engine's skeleton look.
Some things are unfortunately hardcoded in Blender’s core like the visibility of parent lines between bones so a simple script couldn't fix everything.
But I think overall look is much cleaner now.

This is a purely visual change: it doesn’t affect bone hierarchy or rotations. If you jump into Edit Mode, you'll see the original view.

Download here: https://github.com/Shockobonss/Jointify/releases/tag/v1.7
